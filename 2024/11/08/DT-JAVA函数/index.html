JAVA函数

什么是函数
函数就是定义在类中的具有特定功能的一段独立的小程序，也称独立的代码块，我们可也通过调用函数来执行相应的代码。

说人话：简单来说，在一个类（一个.JAVA文件）里面，一段有自己存在的的意义，有自己作用的独立代码

下面我们就来说说JAVA函数的语法：

这里有一个知识点：修饰符
修饰符分为：访问控制修饰符 与 非访问控制修饰符

现在我们来看看，访问修饰符

访问控制：顾名思义就是控制访问，用来保护类，变量，方法和结构方法的访问，这里就要讲到在JAVA里面支持4种不同的访问权限

1.default
默认，什么也不写，在同一个包内可以访问,可以使用在：类，接口，变量，方法
2.private
私有的，在同一个类可见，可以使用在：变量方法。说到这就有一个注意点了:private 不能修饰类（外部类）
3.public
公共的，所有类可见，可以使用在：类，接口，变量，方法
4.portected
对同一包内的类和所有子类可见，可以用在：变量，方法。注意的是：protected 不能修饰类（外部类）
