1.数据类类型的划分

数据类型在语言程序的构成要素中有着重要的地位。JAVA的数据类型可以分为基本数据类型和引用数据类型，后面我会写出来。

原始的数据类型也称为基本数据类型，包括了最基本的byte,short,int,long,float和dauble等类型。

另外一种呢就是引用数据类型了，它很特殊，它是一种特殊方式指向变量的实体，这种机制类似于C++的指针。哎，不要想太多，这类变量在声明时不会分配内存，但是必须另外进行开辟内存空间的操作，比如字符串与数组均属于这种数据类型。

下面说到的就是JAVA中所规定的8种数据类型，原本是为了各位方便理解，想着以思维导图的方式呈现的，但是呢，出来点小意外，没能做的出来，以下就将就着看吧，后期我在慢慢改。

数据类型

- 类 (clas)

下面就该我们的整数类型了

整数类型

整数在小学一年级的数学就开始认识了，比如0.1.2.3.4.5…..这些的整数，但现在就多了个-1.-2.-3.-4……的负整数。

当数据不带有小数或分数时，都可哟声明为整数变量，如（-∞，0），（0，+∞），但是得注意上面的表格里也写了可表示的最大值，不真正的无穷大。在JAVA中，整数数据类型可分为long，int，short及byte4种，long为64位，也就是8个字节，范围看表格；int为32位，也就是4个字节，范围……；如果数值越小，可以声明short（短整数）或byte（位），以节省空间。

怎么声明一个整型变量呢，比如我们声明一个短整型变量sum，这里的sum指的是我们自己给这个变量取的变量名，话不多说，直接开始教程，如下：

shout sum； //声明sum为短整型

经过声明后，JAVA就会在可使用的内存空间中寻找一个占有两个字节的块供sum变量使用，同时，这个变量只能在可表示的数据的范围之内。

在JAVA中，整型变量常量，其类型的默认的就是int型，所以尽量在声明常量时不要超过int数据类型的范围。

要是想知道如果超出了数据范围出现的问题，我也在这里简单的写一个，如下： 

class abab{
    public static void main(String[] args){
        int num=99999999999999999999999999;   //定义整型变量，错误，超出长度范围
    }
}

编译结果;

"message": "The literal 99999999999999999999999 of type int is out of range ",

然后就会出现下面讲的数据溢出；

数据溢出

溢出就相当于超出某个范围，所以这里就相当于整数的数据超大小超出了可以表示的范围，而程序中又没有做数值范围的检查，这个整形的变量所输出的值就会发生絮乱，且不是我们预想的结果。

如果要想声明一个整数，并把它赋值为整型所可以表示的范围的最大值，可以将它分别加1及加2
如： 

class abab{
    public static void main(string[] args){
        int max = lnteger.MAX_VALUE;
        System.out.println("整型的最大值:"+max);
        System.out.println("整型最大值+1："+（max+1）)；
        System.out.println(”整型最大值+2：“+（max+2）)；
    }
}

运行的结果：
整型的最大值：2147483647
整型最大值+1：-2147483648
整型最大值+2：-2147483647

看到结果是不是好奇怎么加2值和原来一样了，那是因为最大值加上1时，结果会变成表示范围中的最小值；
当最大值加上2时，结果会变成可表示范围中次小的值。什么是次小值呢，就是说这些整型变量的行为就像里程表，如果超越了限制，其值将为另一端的取值。
