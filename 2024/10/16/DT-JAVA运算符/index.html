DT_JAVA运算符

JAVA运算符

1.运算符的分类

（1）按操作数的个数分类

单目运算符：只需要一个操作数参运算。
双目运算符：需要两个操作数参与运算。
三目运算符：需要三个操作数参与运算。

（2）按运算符功能来分，基本运算有以下几类：

算术运算符：+，-，*，/，++，– 。
关系运算符：>，<，>=，>=，==，！= 。
逻辑运算符：！，&&，||，&，| 。
位运算符：>>，<<，>>>，&，|，^，~ 。
赋值运算符：=,扩展赋值运算符，如+=,/=等
条件运算符：？：
其他：·，[]，instanceof,()等

以下就是我们所用的运算符了

（1）算术运算符

最基础也是最简单的几种

运算符 作用 如…
+ 加 1+1=2
- 减 2-1=1
* 乘 10*10=100
/ 除 4/2=2
% 取余 3%2=1
++ 自增：被使用后操作数加1 1++=2
– 自减：被使用后操作数减1 1–=0
++ 自增：操作值先加1再进行使用 ++1=2
– 自减：操作值先减1再进行使用 –1=0
（2）关系运算符

设变量a为5，变量b为10

运算符 说明 示例
== 检查两个操作数的值是否相等，如果相等则为真 (a==b)不为真
!= 检查两个数是否相等，如果不相等就为真 (a!=b)为真
>= 检查a是否大于等于b，如果是就为真 (a>=b)不为真
<= 检查a是否小于等于b，如果是就为真 (a<=b)为真
> 检查a是否大于b，如果是就为真 (a>b)不为真
< 检查a是否小于b，如果是就为真 (a<b)为真

说到这里，有一点要注意 =和==不能混用一个是赋值运算符，一个是等于运算符，（a=b）这样写出来程序会给出报错

（3）位运算符

假设变量a为60，变量b为13
用二进制表示为：00111100，0001101

运算符 说明 示例
& 按位与，如果同时存在与两个操作数中，二进制AND运算符复制一位到结果中 (a&b)将得到12，也就是00001100
| 按位或，如果存在于任意操作中，二进制OR运算符复制一位到结果中。 （a\b）将得到61，即为0011 1101
^ 如果存在于其中一个操作数中但不同时存在于两个操作数中，二进制或运算符复制一位到结果中 (a^b)将得到49，也就是00110001
~ 二进制补码运算符是一元运算符，具有”翻转”位的作用 (~a)将得到-61,也就是11000011，二的补码格式，带符号的二进制数
<< 二进制左移运算符，左操作数的值向左移动右操作数指定的位数 a<<2得到240，也就是11110000
>> 二进制右移运算符，左操作数的值向右移动右操作数指定的位数 a>>2得到15，也就是00001111
(4)逻辑运算符

我们最常用的三个莫非就是;&&，||，!。

运算符 说明 示例
&& 逻辑与，若两个关系式都为0(真)，则条件为真 (a<0&&b>0)为假
|| 逻辑或，若两个关系式有一个为0(真)，则条件为真 (a<0||b>0)为真
! 逻辑否，这个将两个关系式的结果逆转，原本为真的加上这个会变为否 !(a<0&&b>0)为真

注意
逻辑运算符可以针对结果为布尔值得表达式进行逻辑运算，如：5>3&&1=0的结果为真（true）。
运算符“&”和“&&”都表示与操作，当且仅当运算符两边操作数都为true时，其结果才为true，否者结果为假（false）。
“&“和”&&“的区别：在使用”&“进行运算时，无论左边为真还是假，右边的表达式都会进行运算。但如果使用”&&“进行运算时，当左边为假时，右边的表达式就不会进行运算，因此”&&“被称为短路与。
“|”和”||“都表示或操作，当运算符任何一边的操作数或表达式为true时，其结果为true，当两边的值都为false时，其结果才会为false，同与操作类似。”||“表示短路或，当运算符的左边为true时右边的表达式才不会进行运算。

（5）赋值运算符
运算符 说明 示例
= 最简单的一个，他就叫赋值运算符，将右边的操作数赋给左操作数 a=10
+= 加赋值运算符，将右操作数加上左操作数并赋值给左操作数 a+=10
-= 减赋值运算符，将右操作数减去左操作数并赋值给左操作数 a-=10
*= 乘赋值运算符，将右操作数乘以左操作数并赋值给左操作数 a*=10
%= 取模赋值运算符，取两边操作数的模并赋值给左操作数 a%=10
<<= 左移赋值运算符 a<<=2等同于a=a<<2
>>= 右移赋值运算符 a>>=2等同于a=a>>2
&= 换位与赋值运算符 a&=2等同于a=a&2
^= 换位异赋值运算符 a^=2等同于a=a^2
| 换位或赋值运算符 a|=2等同于a=a|2

一目 二目 三目运算符

一目运算符就是只需要一个操作数就可以的运算符，例如：自增运算符，自减运算符
二目运算符就是需要两个操作数才可以的运算符，例如：赋值运算符，加法运算符
三目运算符就是需要三个操作数才可以的运算符，例如： ? :
